Not Mario | 27/11/2021

This was my first project using the OpenGL based ASGE library, the recreation of World One, Level One in Mario.

Deep Dive video into my thought process behind the code. You can check out a purely gameplay video here.

Despite getting top marks for this project, its definitely not one I am proud of. There are so many systems which could have been improved or created in different ways. However, the advantage of going so wrong with everything is that you can learn from everything! My failings in this project taught me a lot, and I was able to use those lessons to greatly improve my future code.
