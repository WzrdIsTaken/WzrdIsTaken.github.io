Telekinesis Project | 10/01/2022

In this university project we had to implement some mechanics from Remedy's Control in two game engines. I chose Jesse's telekinetic abilities - namely levitation, launch and shield.

I used Unity as it was the engine I was most comfortable in, and Unreal because I needed to learn it!

The aim was not only to create a solid mechanic, but also a pipeline for a designer so that they could easily make changes to the system.
