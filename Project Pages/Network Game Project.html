C++ Network Game | 12/02/2022

In this university assignment we had to create a multiplayer turn based game in C++ with a client server architecture.

This project was quite a challenge due to the complexity of working with networking, and the scope of the task we attempted - but I am very proud with the result! I used the lessons I learned while making the NotMario project to improve the quality and structure of my codebase.

This project also includes my custom wrapper around ASGE (the game engine we used), which takes inspiration from engines like Unity and Godot as it primarily implements the concept of scenes.
